⚠️ One-Time Payments Collection Reset

This deletes every document currently in the payments collection, then rebuilds it from scratch using submissions where paymentVerified === true.

Run this once, only after you've deployed the patched admin.html (the one that writes paidAt and creates a matching payment doc on every Paid toggle). Running it again later is harmless but unnecessary.

This action cannot be undone. Make sure you're pointed at the correct Firebase project before clicking.
